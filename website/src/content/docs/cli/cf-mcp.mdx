## Overview

`cf mcp` generates a [Model Context Protocol](https://modelcontextprotocol.io/) (MCP) server configuration for your learning documentation. Use this to integrate your docs with other LLM clients like ChatGPT, Claude web, or other AI tools.

## Usage

```bash
cf mcp [path]
```

The default path is `<project>/docs/learn/`.

## Output

The command prints a JSON configuration object that you can paste into your MCP client setup. This enables those tools to access and search your learning documentation in real time.
